# Test Fixtures

All fixtures in this directory are synthetic and may be redistributed under
the repository's MIT license.

- `Activities/synthetic-run.fit` is generated by
  `scripts/generate-fixtures.py`. It contains no real athlete, device, or GPS
  data.
- `Icons/` contains minimal authored SVG inputs used only to verify rendering
  behavior.
- `VisualSnapshots/` contains renderer output generated from synthetic
  activity data.

Regenerate the FIT fixture with:

```sh
./scripts/generate-fixtures.py
```

Regenerate visual snapshots with:

```sh
UPDATE_VISUAL_SNAPSHOTS=1 ./scripts/visual-test.sh
```

Visual snapshot changes require human review.
